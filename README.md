# Read Before Playing
Play the game by:
1. Downloading and unzipping this to a folder
2. Open the whole folder in Visual Studio Code
3. Open the index.html file and open it using LiveServer addon from VSC
4. Enjoy playing the game~!

Controls:
W - Jump
A - Run Left
D - Run Right
